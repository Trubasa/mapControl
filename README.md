# map-control

基于 vue2 封装 fabricjs 的组件

## 使用

1. npm 包安装

```安装
npm install map-control
```

```在main.js中注册
import mapControl from "map-control";
Vue.use(mapControl);
```

```在vue文件中使用
<map-control></map-control>
```

2. 复制源码使用

## 开发

```bash
# 安装依赖
npm install

# 开发
npm run dev

# 打包
npm run build

# 发布
npm public
```
